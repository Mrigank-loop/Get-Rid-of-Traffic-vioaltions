# IntelliTraffic AI

AI-powered traffic violation monitoring system using Computer Vision and YOLOv8.

## Features
- Night Vision Enhancement
- Vehicle & Pedestrian Detection
- Helmet Compliance Monitoring
- Triple Riding Detection
- Wrong Side Driving Detection
- Waterlogging Detection
- License Plate OCR
- AI Traffic Dashboard

## Tech Stack
- Python
- OpenCV
- YOLOv8
- EasyOCR
- Streamlit

## Project Architecture
Video Input
↓
Image Preprocessing
↓
Object Detection
↓
Violation Detection
↓
OCR
↓
Evidence Generation
↓
Dashboard & Analytics
